Featured Project

KaiForge

The ultimate analytical companion for table tennis professionals. KaiForge transforms raw match footage into a strategic roadmap.

C++ Real-time Analytics Supabase Computer Vision

Project Philosophy

KaiForge bridges the gap between intuition and raw data. We process high-speed ball trajectories to offer a "forge" where technique is refined through evidence.

Core Functionality

Utilizing frame-by-frame analysis, the system tracks the ball at speeds exceeding 100km/h, providing heatmaps and momentum tracking.

Deep Tech Stack

C++ Backend: Handles the core mathematical engine with zero-latency overhead for high-frequency video processing.

Supabase: Manages the central nervous system, syncing player profiles and historical trends via PostgreSQL.
